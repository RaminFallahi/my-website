TORONTO LIBRARIES

This is a simple Toronto Libraries application that uses Node.js and JavaScript to extract information from the http://www.torontopubliclibrary.ca/data/library-data.kml API.

The application displays a list of all the libraries in Toronto and allows users to click on each library to view its related information such as its address and phone number.

The data is stored in an XML file format, and the program uses Node.js to fetch and parse the data from the API.

This application is a useful tool for anyone looking to find information about Toronto's public libraries.
