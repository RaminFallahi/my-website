TRAKT MOVIES API

This simple movie application is built using Node.js, .PUG, and JavaScript, and it extracts information from the TRAKT movie website API.

The application displays a list of movies that people are currently watching live on the TRAKT website.

This project showcases the web development skills of the creator and highlights their ability to use APIs to extract data for use on their own website.
