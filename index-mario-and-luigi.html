MARIO AND LUIGI.

Mario and Luigi's thrilling race comes to life in this interactive animation built using HTML, CSS and JavaScript.

The animation features Mario and Luigi racing each other on a vibrant road, and a start/pause button allows the user to control their movement.

While the user can control Mario and Luigi's movement, other elements on the page, such as the clouds and mushrooms, move independently to give the scene a more realistic feel.

The speed, duration and transforms used in the animation create a fun and engaging experience for the user.

Also, the whole page and its elements are made using CSS.
