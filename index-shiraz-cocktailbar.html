SHIRAZ COCKTAIL BAR.

Shiraz website built in Node.JS, MongoDB, and .PUG is a robust and dynamic platform that extracts data from a MongoDB database.

The website uses Node.JS as the server-side technology, allowing for efficient data processing and management.

The website's frontend is developed using .PUG, a templating engine that enables dynamic and modular web development.

The site's design is implemented using CSS, ensuring a modern and visually appealing user interface. By extracting data from a MongoDB database, the website is equipped to handle large volumes of data efficiently.

Overall, the website is an excellent example of a modern web application developed using the latest technologies in web development.
